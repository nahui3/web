# Документация проекта

## Описание

Простой прототип React приложения, созданный с использованием Vite.

## Структура проекта

```
web/
├── src/
│   ├── App.jsx          # Главный компонент приложения
│   ├── App.css          # Стили для App компонента
│   ├── main.jsx         # Точка входа приложения
│   └── index.css        # Глобальные стили
├── index.html           # HTML шаблон
├── vite.config.js       # Конфигурация Vite
└── package.json         # Зависимости и скрипты
```

## Технологии

- React 18
- Vite 5

## Скрипты установки

В корне проекта находятся скрипты для автоматической установки Node.js и npm:

- `install.sh` - для Linux/macOS
- `install.bat` - для Windows

### Что делают скрипты

Скрипты автоматически:

1. **Проверяют наличие Node.js и npm**
   - Проверяют, установлены ли Node.js и npm
   - Проверяют версию Node.js (требуется версия 18 или выше)
   - Если всё установлено и версия подходящая, скрипт завершается

2. **Устанавливают Node.js и npm при необходимости**
   - Если Node.js отсутствует или имеет старую версию, скрипт выполняет установку
   - Использует оптимальный способ установки для вашей операционной системы

3. **Способы установки по ОС:**

   **macOS:**
   - Использует Homebrew (если установлен)
   - Если Homebrew отсутствует, выводит инструкцию по его установке

   **Linux:**
   - Ubuntu/Debian: использует apt с репозиторием NodeSource
   - CentOS/RHEL: использует yum с репозиторием NodeSource
   - Fedora: использует dnf с репозиторием NodeSource

   **Windows:**
   - Использует Chocolatey (если установлен)
   - Если Chocolatey отсутствует, скачивает и устанавливает официальный установщик с nodejs.org

4. **Проверяют результат установки**
   - После установки проверяют, что Node.js и npm доступны
   - Выводят версии установленных компонентов
